To create user there you should be super admin first

.. image:: /image/01.jpg
    :alt: Login page
    :target: https://www.nyefpokhara.org/admin
    :class: with-shadow
    :scale: 50

go to `Create User`_

.. image:: /image/02.jpg
    :alt: Login page
    :target: https://www.nyefpokhara.org/admin/auth/user/add/
    :class: with-shadow
    :scale: 10

Fill the Form (username/email and password)
